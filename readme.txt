== Description ==

Integrate Mautic with your Give donation forms. Add donors to Mautic segment when they donate to your Cause with Give - WordPress Donation Plugin.

In order to use this plugin, First you need to install
[AutomatePlus - Mautic for WordPress](https://wordpress.org/plugins/automate-mautic/) and [Give - WordPress Donation Plugin](https://wordpress.org/plugins/give/)
